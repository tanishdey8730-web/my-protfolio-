# Shantanu Dey — Developer Portfolio (HTML)

A premium, recruiter-focused developer portfolio built with pure HTML, CSS, and JavaScript.

## Files

```
index.html      ← Main page (open this in browser)
css/style.css   ← All styles
js/script.js    ← Interactivity & portfolio data
favicon.svg     ← Site icon
```

## How to Use

Double-click `index.html` or open it in any browser. No build step or npm required.

For local development with live reload, use any static server:

```bash
npx serve .
```

## Features

- Dark/Light mode toggle
- Animated typing hero
- Skills, experience timeline, project showcase
- Project search, filter & modal popups
- GitHub stats section
- Certifications carousel
- Command palette (Ctrl+K)
- AI chat assistant
- Contact form (opens email client)
- Scroll progress, loading screen, back-to-top
- Fully responsive & SEO optimized

## Customize

Edit portfolio content in `js/script.js` → `PORTFOLIO` object at the top of the file.

Add your resume as `resume.pdf` in the project root.

Update contact info, social links, and project URLs in the same file.

## Deploy

Upload all files to GitHub Pages, Netlify, or any static hosting:

- `index.html`
- `css/style.css`
- `js/script.js`
- `favicon.svg`
- `resume.pdf` (optional)
